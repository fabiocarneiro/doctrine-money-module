DoctrineMoneyModule
=====================

Provide ZF2 integration between doctrine and mathiasverraes/money

Installation
------------

Use one or both of doctrine/orm and doctrine/mongodb-odm

```javascript
{
  "require": {
    "zfbrasil/doctrine-money-module": "dev-master"
  }
}
```
Copy one or both of the dist files to your config/autoload dir
